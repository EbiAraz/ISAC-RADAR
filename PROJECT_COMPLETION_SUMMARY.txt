SUPPORT & TROUBLESHOOTING
================================================================================

PROBLEM: Docker not found
SOLUTION: 
  1. Install Docker from docker.com
  2. Run: docker --version
  3. Restart terminal

PROBLEM: Port 5000 already in use
SOLUTION:
  1. Find process: netstat -ano | findstr :5000
  2. Kill process: taskkill /PID <PID> /F
  3. Try again: docker-compose up

PROBLEM: CUDA not found (GPU support)
SOLUTION:
  1. Install NVIDIA drivers
  2. Install CUDA toolkit
  3. Install cuDNN
  4. Restart system

PROBLEM: Model too large
SOLUTION:
  1. INT8 quantization is already applied (45 MB)
  2. Use edge device with GPU
  3. Implement model pruning

PROBLEM: Latency too high
SOLUTION:
  1. Check 5G connection: 5G signal should be >-85 dBm
  2. Reduce image resolution
  3. Enable GPU acceleration
  4. Move hub closer to edge nodes

For more troubleshooting, see: QUICK_START.txt or README_5G_AI_SYSTEM.md
