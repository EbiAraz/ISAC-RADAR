================================================================================
TROUBLESHOOTING
================================================================================

ERROR: "fatal: remote origin already exists"
FIX: You already added the remote. View it:
  git remote -v
  
  If it's wrong, remove it:
  git remote remove origin
  
  Then add the correct one:
  git remote add origin https://github.com/YOUR_USERNAME/ISAC-Radar.git

---

ERROR: "fatal: could not read Username for 'github.com'"
FIX: GitHub Personal Access Token needed:
  1. Go to https://github.com/settings/tokens
  2. Generate new token with "repo" scope
  3. Use token as password when pushing

---

ERROR: "Permission denied (publickey)"
FIX: SSH key issue. Use HTTPS instead:
  git remote remove origin
  git remote add origin https://github.com/YOUR_USERNAME/ISAC-Radar.git
  git push -u origin main
